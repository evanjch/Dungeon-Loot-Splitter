    Logic Explanation

    The application uses an array to store all the loot items. Each item is saved as an object with a name and value. A for loop is used to go through the array to calculate the total loot and to display each item in the loot list. If statements then check that the party size is valid, the loot name is not empty, and the loot value is not negative before adding it or splitting the loot.
